FOODGRAM
This is a Food Ordering Website where users can:

🔑 Create an account and log in
🍽️ Browse the menu
🔎 Check calories & nutrition information for each food item
💳 Place an order and make online payment

Features
✅ User Sign Up / Login
✅ Food Menu with Calories details
✅ Cart & Order Management
✅ Payment Integration
✅ Responsive UI (works on desktop)

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js
Database:MongoDB
Payment Gateway:PayPal
